# Contributing to Your Project

Thank you for your interest in contributing to **Your Project**! Bug fixes, new features, and documentation improvements are all welcome.

---

## Getting Started

1. **Fork** the repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/Artpupser/template
   ```
3. Create a branch:
   ```bash
   git checkout -b feature/issue-N-description
   ```

---

## Development Setup

Requirements: **csharp**

```bash
   npm install
```

---

## How to Contribute

- **Bug fix** — open an Issue first, then create a branch like `fix/issue-N-description`
- **New feature** — discuss in Issues or Discussions before building
- **Docs** — a PR is always welcome without a prior Issue

---

## Coding Guidelines

- Follow **csharp** conventions
- Keep methods small and focused
- Avoid unnecessary dependencies

---

## Commit Messages

```
feat: add new capability
fix: resolve crash on startup
docs: update README
refactor: extract validator to separate module
chore: bump dependencies
```

---

## Testing

Please test on: **Linux, Windows, macOS**

---

## Reporting Bugs

Open an [Issue](https://github.com/Artpupser/template/issues) with:
- OS and version
- Steps to reproduce
- Expected vs actual behavior
